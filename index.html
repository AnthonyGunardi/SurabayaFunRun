Our Mission

SURABAYA FUN RUN 2025: LARI SEHAT, CERIA BERSAMA SURABAYA

Surabaya Fun Run tahun ini tidak hanya ajang lari biasa, tetapi juga membawa dampak positif bagi anak muda untuk lebih produktif dan sehat dengan memberikan dampak positif untuk sekitar.

Dengan event lari ini kami membawa nilai-nilai ESG (Enviromental, Social, and Governance), dimana aspek sosial kesehatan melibatkan masyarakat lokal, komunitas peduli kesehatan diperkuat dengan kolaborasi Mengedukasi peserta tentang pentingnya gaya hidup sehat dan aktif secara fisik. Terlihat dalam komitmen penyelenggara untuk menghadirkan event yang berkelanjutan. Surabaya Fun Run membuktikan bahwa olahraga dalam menjadi katalis positif untuk pentingnya gaya hidup sehat bagi lingkungan sekitar.

Jadilah bagian dari sejarah Surabaya Fun Run 2025! Rasakan atmosfer yang tak terlupakan, di salah satu event lari paling berkesan di Surabaya! Siapkan langkahmu, tantang dirimu, dan mari kita berlari bersama! #SurabayaFunRun2025 #LariSehat #CeriaBersamaSurabaya
